# 3-Minute Demo Script

## Opening

"Hi, I’m presenting SmartStudy AI, a Gemma-powered learning platform built for the Gemma 4 Good Hackathon.

The problem we focused on is that students and self-learners work across fragmented inputs like PDFs, lecture notes, YouTube videos, podcasts, research papers, and code, but most AI tools only help with one small piece of that workflow.

SmartStudy AI brings that full loop together. We use local Gemma inference for the main generation path, LightRAG for grounded document retrieval, and exportable learning artifacts so the output is actually useful after the chat is over." 

## Homepage

"On the homepage, you can see that this is not just one prompt box. It is a full multipage product with study tools, NotebookLM-style tools, and technical workflows. You can also see the live system status for Gemma and LightRAG, which is important because grounded answers depend on the retrieval layer being active." 

## NetSeek

"First, I’ll show NetSeek. Instead of generic AI output, NetSeek performs web research, keeps the source trail visible, and synthesizes the findings into a structured brief. This matters because learning tools should not just sound good, they should stay anchored to evidence." 

## EduTube

"Next is EduTube. A learner can take a YouTube video or transcript and instantly turn it into study material like notes or flashcards. This shows how the same Gemma-based system can transform passive content into active learning artifacts." 

## NeuroRead

"Now I’ll show NeuroRead, which is one of the strongest parts of the platform. I upload source documents, index them into LightRAG, and ask a question. The answer is grounded in the uploaded material, and I can inspect the references. Then I ask a follow-up question, and the system preserves short conversation memory so the interaction feels more natural, more like a real study session instead of isolated one-off prompts." 

## QuizVerse

"Once the learner understands the material, the platform can immediately convert it into active recall. Here in QuizVerse, the same source content becomes an interactive quiz with explanations. So the product is not just answering questions, it is helping the learner practice." 

## AudioOverview

"Finally, for learners who prefer spoken explanations, AudioOverview can turn source material into a podcast-style explanation and even generate MP3 narration. That gives us another output modality without leaving the same platform." 

## Closing

"So the key idea is that SmartStudy AI is not a single-feature demo. It is a Gemma-powered learning workflow: search, ingest, ground, explain, quiz, summarize, narrate, and export.

Gemma is not a bolt-on here. It powers the core experience across the product, and the repository also includes an open-source fine-tuning path to further improve grounded context-following behavior.

That is SmartStudy AI."
