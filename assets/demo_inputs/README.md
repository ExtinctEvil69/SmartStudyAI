# Demo Inputs

These files are prepared to make hackathon demos repeatable.

## Suggested usage

- `netseek_queries.md`
  Use in `NetSeek`
- `edutube_video_urls.md`
  Use in `EduTube`
- `rag_vs_cag_study_packet.txt`
  Use in `NeuroRead`, `QuizVerse`, `PrepMaster`, `Studio`, `MultiSourceSynth`
- `audio_overview_transcript.txt`
  Use in `AudioOverview`
- `microservices_architecture_brief.txt`
  Use in `ArchViz`
- `logictrace_bug_report.txt`
  Use in `LogicTrace`

Use these as stable fallbacks if network-dependent demo flows are unavailable.
